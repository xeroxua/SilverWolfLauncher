# Changelog 📜
รายละเอียดการอัปเดตและประวัติเวอร์ชันของ Silver Wolf Launcher

---

## [1.2.0] - 2026-03-21
### ✨ เพิ่มคุณสมบัติและปรับปรุง UX (Added & UX Polish)
*   **🚫 Single Instance Only**: ป้องกันการเปิดโปรแกรมซ้อนกันหลายหน้าต่างด้วยระบบ Mutex
*   **🔘 Play Button Logic**: ปุ่มเริ่มเกมจะไม่หายไป แต่จะกลายเป็นสีเทา (Disabled) หากยังไม่ได้เลือก Game Path เพื่อความสวยงามและป้องกันการกดเริ่มเกมผิดพลาด
*   **⏳ Immediate Feedback**: เพิ่มแถบ Loading และข้อความสถานะทันทีเมื่อกดเช็กอัปเดต จะได้ไม่รู้สึกว่าโปรแกรมค้าง
*   **⚙️ Settings Toggle**: ปุ่มฟันเฟืองสามารถกดเพื่อ "เปิด" และ "ปิด" หน้าตั้งค่าได้แล้ว (Toggle)

### 🩹 แก้ไขข้อผิดพลาด (Fixed)
*   **💾 Checkbox Persistence**: แก้ไขบัค "Don't ask again" ที่ค่าจะถูกล้างทิ้งทุกครั้งที่เปิดโปรแกรมใหม่
*   **✖️ Exit Prompt Fix**: แก้ไขปุ่ม X ในหน้าต่างแจ้งเตือนปิดโปรแกรม ให้ยกเลิกการปิดและกลับสู่หน้าหลัก (ไม่โดนย่อลง Tray แบบสุ่มสี่สุ่มห้า)
*   **📁 Config System**: ปรับปรุงการบันทึกเวอร์ชันจากการติดตั้งด้วยไฟล์มือถือ ให้ลงใน `config.json` อย่างถูกต้อง

### 🗑️ สิ่งที่ถูกนำออก (Removed)
*   **📖 Tutorials & Guides**: นำส่วนของคู่มือและวิดีโอแนะนำออกตามคำขอ เพื่อความคลีนของตัวโปรแกรม

## [1.1.0] - 2026-03-19
### ✨ เพิ่มคุณสมบัติใหม่ (Added)
*   **🔄 Launcher Auto-Update**: ระบบตรวจสอบเวอร์ชันใหม่จาก GitHub และทำการอัปเดตตัวเองอัตโนมัติ
*   **🛠️ Self-Replacement Logic**: ใช้ระบบ PowerShell เบื้องหลังเพื่อสลับไฟล์ใหม่และรีสตาร์ทโปรแกรม
*   **🎨 Update Notification Bar**: แถบแจ้งเตือนสีส้มสไตล์ HSR เมื่อมีเวอร์ชันใหม่ให้อัปเดต
*   **🏷️ Version Tracking**: เพิ่มไฟล์ `launcher_version.txt` เพื่อช่วยในการตรวจสอบเวอร์ชันภายใน

### 🩹 แก้ไขข้อผิดพลาด (Fixed)
*   **🛡️ Airspace Collision Fix**: ปรับปรุงหน้าต่างแจ้งเตือน (Prompt) ให้แสดงผลทับ WebView2 เสมอ (ไม่จมหายไปข้างหลัง)
*   **📦 Build Optimization**: ปรับปรุงการ Publish ให้รองรับทั้งแบบ Folder (Dev) และ Single-File (Release)

## [1.0.0] - 2026-03-18
### ✨ คุณสมบัติเริ่มต้น (Initial Release)
*   **🎨 Premium UI**: อินเทอร์เฟซแบบ Glassmorphism (Honkai: Star Rail Style)
*   **🛡️ Process Watchdog**: ระบบเฝ้าระวังและรีสตาร์ท PS/Proxy อัตโนมัติ
*   **🌐 Integrated Tools**: รวม SR Tools และ Guides ไว้ใน WebView2
*   **🌙 Tray Support**: ระบบย่อโปรแกรมลง Tray และเมนูคลิกขวา
